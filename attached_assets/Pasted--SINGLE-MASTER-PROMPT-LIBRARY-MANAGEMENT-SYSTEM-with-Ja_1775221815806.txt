### ARTIFACT 15: Category Validation Utility Class
**Filename:** `com/lms/util/CategoryValidator.java`

**Content:**
```java
public class CategoryValidator {
    private static final Map<MainCategory, Set<String>> VALID_SUBCATEGORIES = Map.of(
        MainCategory.COMPETITIVE_EXAM, Set.of("SSC", "UPSC", "CAT", "GATE", "NEET", "JEE", "BANKING", "RAILWAY", "OTHER_EXAM"),
        MainCategory.COLLEGE, Set.of("ENGINEERING", "MEDICAL", "COMMERCE", "ARTS", "LAW", "MANAGEMENT", "RESEARCH"),
        MainCategory.SCHOOL, Set.of("K_5", "GRADE_6_8", "GRADE_9_10", "GRADE_11_12"),
        MainCategory.COMIC, Set.of("SUPERHERO", "MANGA", "GRAPHIC_NOVEL", "HUMOR"),
        MainCategory.HISTORY, Set.of("ANCIENT", "MEDIEVAL", "MODERN", "WORLD", "REGIONAL"),
        MainCategory.NON_FICTION, Set.of("BIOGRAPHY", "SCIENCE", "SELF_HELP", "TRAVEL", "BUSINESS", "HEALTH"),
        MainCategory.FICTION, Set.of("LITERARY", "SCI_FI", "FANTASY", "MYSTERY", "ROMANCE", "HORROR"),
        MainCategory.OTHER, Set.of("MAGAZINE", "JOURNAL", "REFERENCE", "AUDIOBOOK")
    );
    
    public static void validate(MainCategory main, String subcategory) {
        if (!VALID_SUBCATEGORIES.getOrDefault(main, Set.of()).contains(subcategory)) {
            throw new InvalidCategoryException("Subcategory '" + subcategory + 
                "' is not valid for main category '" + main + "'");
        }
    }
    
    public static Set<String> getSubcategories(MainCategory main) {
        return VALID_SUBCATEGORIES.getOrDefault(main, Set.of());
    }
}
```
